## VM Testen
- Is het IP-adres correct ingesteld?
```bash
ip address
```
  - Dit moet teruggeven: 172.16.1.70

- Is de FQDN correct ingesteld?
```bash
hostname
```
  - Dit moet teruggeven: quebec1.green.local
